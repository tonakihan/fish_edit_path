# fish_edit_path

Interactive TUI helper to manage `$fish_user_paths` using [fzf](https://github.com/junegunn/fzf).

![example](example.gif)

## Requirements

- [fish shell](https://fishshell.com/)
- [fzf](https://github.com/junegunn/fzf)

## Installation

### Via curl

```sh
curl -Lo ~/.config/fish/functions/fish_edit_path.fish https://raw.githubusercontent.com/tonakihan/fish_edit_path/main/fish_edit_path.fish
```

### Manual

Download [`fish_edit_path.fish`](https://raw.githubusercontent.com/tonakihan/fish_edit_path/main/fish_edit_path.fish) and place it in your fish functions directory:

```
$HOME/.config/fish/functions/fish_edit_path.fish
```

## Usage

```fish
fish_edit_path
```
